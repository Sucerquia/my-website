MyUtils

MyUtils is a set of tools that you can use for scientific proporsals, mainly related with the field of simulations of molecules, although there are some parts that could be interesting for all the scientific community. This set of tools was created and it is currently maintained by Daniel Sucerquia, PhD student at the Heidelberg Institute for Theoretical Studies (HITS) and the Max Planck Institute for Polymer Research (MPIP).
